# 🚦 도로 표지판 자동 탐지 시스템 (Road Sign Detection System)

본 프로그램은 **YOLOv8** 딥러닝 모델을 기반으로 도로 주행 영상 및 이미지 내의 주요 표지판을 실시간으로 식별하고 분류하는 AI 솔루션입니다. 높은 정밀도와 재현율을 바탕으로 운전자 보조 시스템(ADAS)이나 자율주행 연구에 활용될 수 있도록 설계되었습니다.

## 📝 프로그램 개요 (Overview)
* **프로그램명:** YOLOv8 Road Sign Detector
* **핵심 기능:** 다중 도로 표지판 위치 추적 및 종류 분류
* **탐지 대상:** 4개 클래스 (Speed Limit, Stop, Crosswalk, Traffic Light)
* **적용 분야:** 자율주행 데이터 분석, 블랙박스 영상 분석, 교통 시설물 관리

## ⚙️ 주요 기능 (Key Features)
1.  **고성능 탐지 엔진:** 0.96 이상의 높은 탐지율(mAP@50)로 표지판을 놓치지 않고 포착합니다.
2.  **높은 신뢰성:** 91% 이상의 정밀도(Precision)를 확보하여 오탐지(False Alarm)를 최소화했습니다.
3.  **강건한 환경 대응:** 흐린 날씨나 복잡한 도심 배경 속에서도 신호등과 표지판을 효과적으로 구분합니다.

## 📊 성능 평가 리포트 (Performance Report)
자체 테스트 데이터셋을 통해 검증된 시스템의 실제 성능 지표입니다.

### 1. 종합 성능 요약
| 지표 (Metric) | 점수 (Score) | 설명 |
| :--- | :--- | :--- |
| **정확도 (mAP@50)** | **96.1%** | 객체 탐지 성공률 (매우 높음) |
| **종합 점수 (mAP@50-95)** | **79.7%** | 위치 정밀도까지 고려한 엄격한 기준 |
| **정밀도 (Precision)** | **91.8%** | 탐지한 것 중 정답일 확률 |
| **재현율 (Recall)** | **93.3%** | 실제 표지판 중 찾아낸 비율 |

### 2. 클래스별 세부 성능 (mAP@50-95)
* **🛑 Stop (정지) & 6️⃣0️⃣ Speed Limit (속도제한):** **86~87%**
    * 형태가 고정적이고 특징이 뚜렷하여 거의 완벽에 가까운 인식률을 보입니다.
* **🚶 Crosswalk (횡단보도):** **74.4%**
    * 복잡한 배경 속에서도 준수한 수준으로 횡단보도 표지판을 식별합니다.
* **🚦 Traffic Light (신호등):** **70.9%**
    * 작은 객체 크기와 조명 변화에도 불구하고 70% 이상의 안정적인 식별 성능을 확보했습니다.

## 🚀 시스템 동작 흐름 (Workflow)
프로그램은 데이터 입력부터 결과 출력까지 다음 4단계 프로세스로 동작합니다.

1.  **입력 (Input):** 도로 주행 이미지 또는 비디오 프레임을 로드합니다.
2.  **전처리 (Preprocessing):** 모델 입력 규격(640x640)에 맞춰 리사이징 및 정규화를 수행합니다.
3.  **추론 (Inference):** 학습된 YOLOv8 엔진이 이미지 내 객체의 위치 좌표와 종류를 계산합니다.
4.  **시각화 (Visualization):** 탐지된 객체 위에 Bounding Box와 클래스명, 신뢰도를 표시하여 사용자에게 출력합니다.

### 1.  **결과(이미지 해석):**
* 탐지 자체는 70% 가량 탐지했습니다.
* crosswalk, speedlimit, stop의 경우 너무 작거나 형태가 너무 다르지 않으면 0.9이상의 높은 Score로 맞추는 경향을 보입니다.
* 다만 데이터셋의 한계 때문인지 같은 모양이어도 색상이 다르면 감지하지 못하거나 틀리는 경우가 있습니다.
   * 예) result15 이미지의 경우 trafficlight인데 빨간색만 보고 stop이라 예상하며 Score도 0.87이나 나왔습니다.
* 그리고 trafficlight는 각도가 많이 돌아가 신호등의 색이 거의 보이지 않는 경우에는 인식을 못하는 경향을 보였습니다.

### 2. **결론:**
   * 정면으로 보이는 roadsign의 경우 90%이상의 정확도를 보이는 것 같습니다.
   * 같은 표지판이더라도 색상이 다른 표지판을 추가한 데이터셋을 사용한다면 정확도가 훨씬 상승하지 않을까 예상합니다.
   * trafficlight의 경우 더 다양한 각도의 데이터가 추가된다면 성능이 더욱 좋아질 것 같습니다.
